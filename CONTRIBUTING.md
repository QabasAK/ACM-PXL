# 🤝 Contributing to ACM-PXL

We're so excited you're here! This project is all about making academic life at PSUT more fun and manageable. Here’s how you can join the mission as a **Code Cadet**.

## Getting Started

1.  **Fork the repo:** Create your own copy of the repository.
2.  **Clone it:** `git clone https://github.com/YourUsername/ACM-PXL.git`
3.  **Setup:** Follow the `README.md` to get Ollama and Streamlit running.
4.  **Create a branch:** `git checkout -b feature/cool-new-pun`

## What can you contribute?

### Knowledge & RAG
Help PXL get smarter!
- Summarize a tricky topic (e.g., *AVL Rotations* or *Big-O proofs*) into a clean PDF or Markdown file.
- Add it to `DSA RAG` or `Algo RAG`.
- Implement the RAG for courses other than Algo & DSA!
- Improve Embeddings!
- **Pro-tip:** Clear headings and bullet points help the AI find answers faster!

### Frontend & UX
- Improve the `Frontend.py` with custom CSS or layout tweaks.
- Add new interactive elements to the `SurvGuide.py`.
- Enhance the charts and graphs in the `ABOUT` section.

### Logic & Personality
- Add new puns or quirky analogies to `BEfun.py`.
- Tweak the `ask.py` prompt template to improve answer quality.
- Suggest new model parameters to save more RAM.

## Rules of the Road
- **Be Cheerful:** PXL is a friendly mascot. Keep code and comments helpful and positive!
- **Respect Privacy:** Never commit personal data or API keys.
- **Test First:** Run the app locally to make sure your changes don't break PXL's "brain."

## After your PR is merged...
You'll be added to the **[HALL_OF_FAME.md](HALL_OF_FAME.md)** and earn the title of **Official PXL Contributor**! 

*Questions? Reach out to the ACM team or open an Issue here on GitHub.*
